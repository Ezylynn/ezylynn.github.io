YAML (YAML Ain't Markup Language)

Definition: A human-readable data serialization language used for config files and data exchange.

Applications: - Allow humans to easily read or create config files - Typically store things a program would need at run time, without needing to hard code the values

ELIF: A general file format to store structured data that's way more readable to humans

Do not confuse with: - YAML is a superset of JSON, technically a YAML parser can understand JSON - YAML is more easily readable by humans while JSON has a more rigid structure

Example:

key: value
list:
  - item1
  - item2

JSON (JavaScript Object Notation)

Definition: A light-weight format for storing and transporting data. Syntactically identical to the code for creating JavaScript objects as it derives from JavaScript

Application: - Often used for light-weight data exchange between APIs and web servers - Storing config settings - Used in security tokens JWTs (JSON Web Tokens) for authentication purposes

ELIF: A way to store data in text in an organized way instead of XML or binary formats

Example: 

{
  "id": 1,
  "name": "Example Item",
  "enabled": true,
  "count": 3,
  "tags": [
    "example",
    "demo"
  ],
  "details": {
    "owner": "Alice",
    "priority": "medium"
  }
}

XML (eXtensible Markup Language)

Definition: A language/file structure designed to store and transport data

ELIF: Basically HTML but it doesn't do ANYTHING, just information wrapped in tags

Application: - XML was designed to carry data, not display them unlike HTML - XML tags and document structure must be defined manually by users - Stores data in plain text format, providing a software & hardware independent way of storing, transporting and sharing data

Do not confuse with: HTML

Static

Definition in Java context: The static members (variable, methods, nested class) belongs to the class itself, not to any specific object (instance) of that class.

Definition in website context: Describe pages with fixed content that appears the same to every user, without any real-time, server side processing or interaction with database

Definition in IP address context: An IP address that does not change frequently. Used in cases of hosting, web servers and secure communications

Base64

Definition: A binary-to-text encoding that uses 64 printable characters, with each character represents 6 bits to maintain integrity of data

ELIF: A way of transforming data into clean, safe data that will make sure data does not get change passing through systems

Application: - Includes upper/lowercase English letters, digits 0-9 & plus (+) and forward slash (/) - Makes data 33% larger in memory usage, should only use when necessary - Does not contain special characters that might be interpret as special commands - Ensure data doesn't get corrupted when systems expect clean readable texts - Note safe for URL

Base32

Definition: A binary-to-text encoding that uses 32 printable chracters, each character are 5 bits in length

ELIF: Same shit like Base64 but less bits used

Application: - Allow data to be passed to a system that accepts single-byte character input, or UTF-8 encoded string input or appended to a URL without being mangled - Use in URLs as Base64 have those troubling symbols and may be treated specially

Parsing

Definition:The process of analyzing a sequence of data (string/bytes) into its components (tokens) for computers to work with reliably

ELIF: Turning some kind of data into another kind of data but with structures

Application: - Apply grammar to tokens to build structure

Do not confuse with: - Parsing is not transforming one thing into another, that's the job of the compiler. Parsing is one of the step to compile a program - Parsing does not extract meaning from a text, that's semantic analysis, a step of the compiling process

Tokenization

Definition: Breaking down stream of characters into smaller, meaningful units called tokens

ELIF:

Application: - After broken down into tokens --> pass through parser - Part of the broader process of processing raw inputs into structured meaning that your program can work with - Use case in compilers, intepreters, data deserialization, protocol message decoding

ELIF: Establish a tunnel between VPN server and you, and send the traffic through there, which the VPN will forward to your intended destination

Application: - Can allow you to access restricted content, depending where VPN server is located - Self-host VPNs, allow to access network over the internet securely - If on HTTP, then allow to protect sensitive data and personal informationAllow to work remotely over the internet
